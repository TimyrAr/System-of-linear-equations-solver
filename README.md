# SLaESolver – Автоматический решатель разреженных СЛАУ
Модуль автоматически выбирает оптимальную пару «метод + предобусловливатель» 
на основе анализа свойств матрицы и доступной вычислительной среды (CPU, GPU, MPI).
## Возможности
- Анализ матрицы: симметричность, положительная определённость, блочная структура.
- Назначение решателя согласно заданной таблице:
  - SPD → CG + IC
  - Симметричная неопределённая → GMRES(200) + ILU
  - Общая несимметричная → BiCGSTAB + ILUT
  - Блочная → GMRES(200) + блочный предобусловливатель
- Параллельное выполнение:
  - CPU (SciPy)
  - GPU (CuPy)
  - Распределённые вычисления (MPI + PETSc)
- Постобработка: относительная невязка и ошибка (при наличии эталонного решения).
## Зависимости
Обязательные:
- Python ≥ 3.10
- numpy
- scipy ≥ 1.11
Опциональные:
- mpi4py + petsc4py (для MPI)
- cupy-cudaXXX (для GPU, версия зависит от вашего CUDA)
